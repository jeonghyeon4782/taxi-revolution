## 어떤 기대 효과가 있을까?
- **안전성 향상**: 택시 정보와 위치 공유를 통해 사용자의 안전을 강화하고 범죄 예방에 기여합니다.
- **경제적 이점**: 카풀을 통해 택시 요금을 절감하고, 장거리 이동시에도 금전적 부담을 덜 수 있습니다.
- **편리한 이용**: 사용자들은 주변 택시 탑승자의 위치를 쉽게 알 수 있고, 탑승 중 긴급한 상황에서도 SOS버튼을 통해 빠른 대처가 가능합니다.

</br>
</br>

## 👊 주요 기능

👤 **회원**
- 회원가입, 회원탈퇴, 로그아웃
- JWT 토큰 기반 로그인

🚕 **탑승 및 하차**
- 보호자 번호 등록
- 문자 주기 설정
- 주변 택시 승강장 조회
- 탑승 기록 조회
- 택시 번호판 인식
- SOS 긴급 문자 전송(112, 119)
- 주기적으로 보호자에게 현재 위치 문자 전송

🍀 **카풀**
- 카풀 게시글 작성, 수정, 조회, 삭제
- 최신 순, 위치 순, 거리 순 게시글 정렬
- 한글 주소로 위치 검색
- 카풀 멤버 추방
- 댓글 작성 및 삭제
